Fun Facts about data mining




Every day we create 2.5 quintillion bytes of data or 2,500,000,000,000,000,000 bytes.

If we stored all this data on two terabyte hard drives the tower would be nine kilometers tall.

In 1992 we created 100 gigabytes of data in a day, ten years later in 2002 we created the same amount of data in just one second.

We create 30 thousand gigabytes of data every single second.
